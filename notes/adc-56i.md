# Text Representation Truncation - Implementation Plan

## Executive Summary

This document outlines the implementation requirements for adding text truncation to the aide-de-camp application. The feature addresses the need to limit display length for long text representations (summaries and data) on the canvas while preserving full content for other uses (audio mode, data export).

## Current State Analysis

### Where Text is Stored
**Database**: `data/session.db` → `results` table
- `summary` (TEXT): Conversational summary generated by LLM
- `data` (TEXT/JSON): Structured result data

### Where Text is Rendered
**File**: `src/canvas/index.html`
- `.result-summary` (line 452-456): Displays summary text
- `.result-data` (line 460-462): Displays preformatted JSON data

### Where Text is Generated
**File**: `src/synthesize/strand.py`
- `SynthesizeResult.summary` (line 53): Generated by LLM call
- `SynthesizeResult.data` (line 52): Structured data from fetch sources

**File**: `prompts/synthesize.md`
- Instructs LLM to generate "2-3 sentence narration" (line 30) — **no hard character limit enforced**

### Existing Constraints
- `max_height: 150px` with `overflow-y: auto` for result data (CSS, line 201) — **scroll-based, not truncation**
- No character limits on summary text
- JSON data can be arbitrarily large

## Implementation Plan

### Option A: Application Layer Truncation (Recommended)

**Rationale**: Application-layer truncation allows for:
- Smart truncation at word/sentence boundaries
- Context-aware "..." indicators
- Full-text preservation in database
- Different truncation rules per surface type (canvas vs audio)

#### 1. Add Truncation Utility Module

**File**: `src/utils/text.py` (new file)

```python
"""
Text processing utilities for aide-de-camp.
"""

import unicodedata
from typing import Optional


# Default truncation limits (configurable)
SUMMARY_MAX_CHARS = 280  # ~2-3 sentences
SUMMARY_MAX_SENTENCES = 3
DATA_JSON_MAX_CHARS = 1000  # For display purposes


def truncate_at_boundary(
    text: str,
    max_chars: int,
    respect_sentence: bool = True,
    ellipsis: str = "..."
) -> str:
    """
    Truncate text at a natural boundary (word or sentence).
    
    Args:
        text: Input text to truncate
        max_chars: Maximum character limit
        respect_sentence: If True, truncate at sentence boundary when possible
        ellipsis: String to append when truncated
    
    Returns:
        Truncated text with ellipsis if shortened
    """
    if not text or len(text) <= max_chars:
        return text
    
    # Reserve space for ellipsis
    effective_max = max_chars - len(ellipsis)
    
    if respect_sentence:
        # Try to truncate at sentence boundary
        truncated = text[:effective_max]
        last_period = truncated.rfind('.')
        if last_period > effective_max // 2:  # At least halfway through
            return text[:last_period + 1] + ellipsis
    
    # Fallback to word boundary
    words = text[:effective_max].split()
    if len(words) > 1:
        return ' '.join(words[:-1]) + ellipsis
    
    # Last resort: hard truncate
    return text[:effective_max] + ellipsis


def truncate_sentences(text: str, max_sentences: int, ellipsis: str = "...") -> str:
    """
    Truncate text to a maximum number of sentences.
    
    Args:
        text: Input text to truncate
        max_sentences: Maximum number of sentences to keep
        ellipsis: String to append when truncated
    
    Returns:
        Text truncated to sentence limit
    """
    if not text:
        return text
    
    # Split by sentence boundaries (. ! ?)
    sentences = []
    current = []
    for char in text:
        current.append(char)
        if char in '.!?':
            sentence = ''.join(current).strip()
            if sentence:
                sentences.append(sentence)
            current = []
    
    # Add any remaining text as partial sentence
    if current:
        remaining = ''.join(current).strip()
        if remaining and len(sentences) < max_sentences:
            sentences.append(remaining)
    
    if len(sentences) <= max_sentences:
        return text
    
    return ' '.join(sentences[:max_sentences]) + ellipsis


def truncate_json_display(json_str: str, max_chars: int = DATA_JSON_MAX_CHARS) -> str:
    """
    Truncate JSON string for display purposes.
    
    Preserves JSON structure by truncating at closing braces/brackets.
    
    Args:
        json_str: JSON string to truncate
        max_chars: Maximum character limit for display
    
    Returns:
        Truncated JSON string with placeholder for omitted content
    """
    if not json_str or len(json_str) <= max_chars:
        return json_str
    
    # Simple heuristic: truncate at a closing bracket
    truncated = json_str[:max_chars]
    
    # Find last closing bracket/brace
    for i in range(len(truncated) - 1, -1, -1):
        if truncated[i] in '}]':
            return truncated[:i + 1] + '\n  ... (truncated)'
    
    # Last resort: hard truncate with marker
    return truncated + '...'


def count_grapheme_clusters(text: str) -> int:
    """
    Count user-perceived characters (grapheme clusters).
    
    Handles emoji, combining characters, and other Unicode complexities.
    
    Args:
        text: Input text
    
    Returns:
        Count of grapheme clusters
    """
    # Simple implementation using unicodedata normalization
    # For full grapheme support, consider `grapheme` library
    normalized = unicodedata.normalize('NFC', text)
    count = 0
    prev_category = None
    
    for char in normalized:
        category = unicodedata.category(char)
        # Skip combining marks (categories Mn, Mc, Me)
        if category.startswith('M'):
            continue
        count += 1
    
    return count


def safe_truncate(text: str, max_chars: int) -> str:
    """
    Truncate text safely, handling Unicode edge cases.
    
    Args:
        text: Input text (may contain emoji, combining characters)
        max_chars: Maximum grapheme clusters to display
    
    Returns:
        Truncated text with ellipsis if needed
    """
    if not text:
        return text
    
    # Count graphemes instead of code points
    cluster_count = count_grapheme_clusters(text)
    
    if cluster_count <= max_chars:
        return text
    
    # Use boundary-aware truncation
    return truncate_at_boundary(text, max_chars * 2, respect_sentence=False)
```

```javascript
// Text truncation helpers
const TRUNCATION_LIMITS = {
    summary_chars: 280,
    data_chars: 1000,
};

function truncateText(text, maxChars, respectSentence = true) {
    if (!text || text.length <= maxChars) return text;
    
    const effectiveMax = maxChars - 3; // Reserve space for "..."
    
    if (respectSentence) {
        // Try to truncate at sentence boundary
        const truncated = text.substring(0, effectiveMax);
        const lastPeriod = truncated.lastIndexOf('.');
        if (lastPeriod > effectiveMax / 2) {
            return text.substring(0, lastPeriod + 1) + '...';
        }
    }
    
    // Fallback to word boundary
    const words = text.substring(0, effectiveMax).split(/\s+/);
    if (words.length > 1) {
        return words.slice(0, -1).join(' ') + '...';
    }
    
    // Hard truncate
    return text.substring(0, effectiveMax) + '...';
}

function truncateJsonDisplay(jsonStr, maxChars = TRUNCATION_LIMITS.data_chars) {
    if (!jsonStr || jsonStr.length <= maxChars) return jsonStr;
    
    // Try to truncate at a closing bracket
    const truncated = jsonStr.substring(0, maxChars);
    for (let i = truncated.length - 1; i >= 0; i--) {
        if (truncated[i] === '}' || truncated[i] === ']') {
            return truncated.substring(0, i + 1) + '\n  ... (truncated)';
        }
    }
    
    return truncated + '...';
}

function escapeAndTruncate(text, maxChars, respectSentence = true) {
    return truncateText(escapeHtml(text), maxChars, respectSentence);
}
```

```python
"""
Configuration settings for aide-de-camp.
"""
from dataclasses import dataclass
from pathlib import Path


@dataclass
class TruncationConfig:
    """Text truncation configuration."""
    
    # Summary truncation
    summary_max_chars: int = 280
    summary_max_sentences: int = 3
    summary_truncate_at_sentence: bool = True
    
    # Data display truncation
    data_json_max_chars: int = 1000
    
    # Whether to respect grapheme clusters (emoji-aware)
    unicode_aware: bool = True


# Default configuration
DEFAULT_TRUNCATION = TruncationConfig()


def get_truncation_config() -> TruncationConfig:
    """Get the current truncation configuration."""
    return DEFAULT_TRUNCATION


def load_truncation_config_from_file(config_path: Path) -> TruncationConfig:
    """
    Load truncation configuration from a YAML file.
    
    This allows runtime reconfiguration without code changes.
    """
    import yaml
    
    with open(config_path) as f:
        config_data = yaml.safe_load(f) or {}
    
    truncation_data = config_data.get('truncation', {})
    return TruncationConfig(
        summary_max_chars=truncation_data.get('summary_max_chars', 280),
        summary_max_sentences=truncation_data.get('summary_max_sentences', 3),
        summary_truncate_at_sentence=truncation_data.get('summary_truncate_at_sentence', True),
        data_json_max_chars=truncation_data.get('data_json_max_chars', 1000),
        unicode_aware=truncation_data.get('unicode_aware', True),
    )
```

```python
@dataclass
class TopicCard:
    """A topic card for the canvas - includes staleness info."""
    topic: Topic
    latest_result: Optional[dict] = None
    staleness_seconds: int = 0
    staleness_level: str = "fresh"
    
    def to_dict(self, use_truncation: bool = True) -> dict:
        """
        Convert to dictionary, optionally truncating text fields.
        
        Args:
            use_truncation: If True, apply truncation to summary and data
        
        Returns:
            Dictionary representation of the topic card
        """
        from ..utils.text import truncate_at_boundary, truncate_json_display
        
        result = {
            "topic": self.topic.to_dict(),
            "staleness": {
                "seconds": self.staleness_seconds,
                "level": self.staleness_level,
            },
        }
        
        if self.latest_result:
            latest = dict(self.latest_result)
            
            if use_truncation:
                # Apply truncation to summary
                if "summary" in latest:
                    latest["summary"] = truncate_at_boundary(
                        latest["summary"],
                        max_chars=280,
                        respect_sentence=True
                    )
                
                # Apply truncation to data JSON representation
                if "data" in latest:
                    import json
                    data_json = json.dumps(latest["data"], indent=2)
                    latest["data_truncated"] = truncate_json_display(data_json, max_chars=1000)
            
            result["latest_result"] = latest
        
        return result
```

## Edge Cases and Considerations

### 1. Unicode and Emoji
- **Issue**: Standard string length counts code points, not user-perceived characters
- **Solution**: Use grapheme cluster counting for accurate limits
- **Dependency**: Consider `grapheme` library for full Unicode 13+ support

### 2. Empty Strings and None Values
- **Issue**: Truncation functions must handle null/empty inputs gracefully
- **Solution**: Early return for falsy values

### 3. JSON Structure Preservation
- **Issue**: Truncating JSON mid-string breaks syntax
- **Solution**: Truncate at closing brackets/braces, append "..."

### 4. Audio Mode Compatibility
- **Issue**: Audio surface needs full text for TTS
- **Solution**: Apply truncation only at canvas rendering layer, preserve full text in database

### 5. Internationalization
- **Issue**: Sentence boundaries vary by language (Chinese uses different punctuation)
- **Solution**: Future enhancement with language-specific boundary detection
